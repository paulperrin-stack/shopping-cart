# Shopping Cart

A mock e-commerce shopping cart built with React.

🔗 **[Live Demo](https://shopping-cart-nine-opal.vercel.app/)**

## Features

- Browse products fetched from [FakeStore API](https://fakestoreapi.com)
- Add items to cart with quantity controls
- Update or remove items from cart
- Live cart count in the navbar

## Built With

- React
- React Router
- CSS Modules
- Vitest + React Testing Library
